# 대한민국 공군 픽토그램

## 사용법
1. 시스템에 `pdf2svg` 를 설치합니다
2. `yarn` 이용해 디펜던시를 받습니다.
3. `yarn start` 로 돌립니다.
4. `target/svg` 폴더에 `svg` 들이 왕창 생성됩니다.
5. 알아서 잘 가져다 쓰시면 됩니다.
